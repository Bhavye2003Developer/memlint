# Contributing to memlint

Thanks for your interest in contributing!

## Getting started

```bash
git clone https://github.com/Bhavye2003Developer/memlint.git
cd memlint
pip install -e ".[dev]"
```

## Running tests

```bash
pytest
pytest --cov=memlint   # with coverage
```

## Making changes

- Open an issue first for anything non-trivial so we can discuss the approach
- Keep changes focused — one thing per pull request
- Make sure all tests pass before opening a PR
- Add tests for any new behaviour

## Reporting bugs

Open an issue at https://github.com/Bhavye2003Developer/memlint/issues with a minimal example that reproduces the problem.
